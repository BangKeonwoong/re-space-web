# 🛑 중요: 실행 전 필독 (Read Me First)

이 프로젝트는 현재 **Google Drive(G:)**에 위치해 있어, 파일 시스템 호환성 문제로 인해 자동 설치가 불가능했습니다.
정상적으로 실행하기 위해서는 **반드시** 아래 단계를 따라주세요.

## 🚀 실행 방법 (3단계)

1.  **폴더 이동**
    *   현재 폴더(`web disign`) 전체를 복사하여 **바탕화면**이나 **내 문서**(`C:` 드라이브)로 옮겨주세요.
    *   *이유: Google Drive는 `npm install`시 발생하는 수천 개의 작은 파일 생성을 차단합니다.*

2.  **터미널 열기**
    *   이동한 폴더로 들어가서, 빈 공간에 `Shift + 우클릭` -> **"PowerShell 창 열기"** (또는 VS Code에서 열기)를 선택합니다.

3.  **명령어 입력**
    *   아래 두 명령어를 차례대로 입력하세요.

    ```bash
    npm install
    # (약 1~2분 소요됩니다)
    ```

    ```bash
    npm run dev
    # (서버가 실행되면 나타나는 http://localhost... 주소를 클릭하세요)
    ```

## 🎨 포함된 기능
-   **Vite + React**: 초고속 개발 환경
-   **Tailwind CSS**: 커스텀 디자인 시스템
-   **프리미엄 UI 컴포넌트**:
    -   `Navbar`: 스크롤 반응형 글래스모피즘
    -   `Hero`: 타이포그래피 애니메이션
    -   `TiltCard`: 3D 호버 이펙트 카드
    -   `BentoGrid`: 모던한 그리드 레이아웃
